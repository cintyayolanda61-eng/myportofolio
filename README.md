# 🌸 Personal Portfolio Website

## Deskripsi

Website ini merupakan portofolio pribadi yang saya buat menggunakan **HTML, CSS, dan JavaScript**. Website ini berisi informasi tentang diri saya, kemampuan yang saya pelajari, serta beberapa proyek yang telah saya kerjakan selama menempuh perkuliahan di bidang Teknologi Informasi.

Website ini juga dirancang agar memiliki tampilan yang sederhana, responsif, dan nyaman digunakan baik melalui perangkat desktop maupun smartphone.

## Fitur

* Halaman Home dengan tampilan yang menarik.
* Halaman About Me yang berisi informasi singkat mengenai diri saya.
* Halaman Skills untuk menampilkan kemampuan dan teknologi yang dipelajari.
* Halaman Portfolio yang berisi beberapa proyek yang pernah dibuat.
* Preview proyek menggunakan video.
* Halaman Contact untuk memudahkan komunikasi.
* Desain responsif sehingga dapat diakses dengan baik di berbagai ukuran layar.

## Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)
* Font Awesome

## Struktur Folder

```plaintext
├── index.html       # Halaman utama
```
